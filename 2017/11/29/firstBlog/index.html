配置github

1.建立Repository
建立与你github用户名对应的仓库，仓库名为【your_user_name.github.io】。
2.创建ssh keys，找到id_rsa.pub复制好

检查是否存在ssh keys,到git bash终端输入命令：ls -al ~/.ssh，若存在就会出现id_rsa id_rsa.pub。

3.为刚刚创建的仓库添加ssh keys，到刚创建的仓库依次点击Setting >> Deply keys >> Add deply key，然后填写title和key

title可以随便填写，Key里面填刚刚id_rsa.pub文件复制好的内容，然后点击Add key即可。

4.测试连接

执行命令：ssh -T git@github.com

出现 You’ve successfully authenticated, but GitHub does not provide shell access.
就OK了

5.配置站点文件_config.yml

在你创建的hexo文件夹中找到_config.yml文件，打开拉到最下面，作如下修改：

deploy:

type: git

repo: https://github.com/username/username.github.io.git

branch: master

username填你自己的github用户名
